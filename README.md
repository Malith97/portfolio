# Malith Ileperuma Portfolio

Minimal multi-page personal portfolio built with Next.js, TypeScript, and restrained Tailwind CSS.

## Design Direction

- Plain Systems / Human Notes
- Editorial and calm, not dashboard-like
- Off-white background, charcoal text, thin borders, generous spacing

## Pages

- Home
- Story
- Work & Education
- Case Studies
- Beyond Work
- Contact

## Content

Local content files are stored in:

- `content/case-studies`
- `content/beyond-work`

Each case study follows this structure:

- Context
- Problem
- Constraints
- Solution
- Tools
- Result
- Lessons Learned

## Development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
